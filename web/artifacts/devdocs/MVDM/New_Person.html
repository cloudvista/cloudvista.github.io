New_Person

New Person

This file contains data on employees, users, practitioners, etc. who were previously in Files 3,6,16 and others.

DHCP packages must check with the KERNEL developers to see that a given number/namespace is clear for them to use.

Field numbers 53-59.9 reserved for Pharm. Nodes and X-ref 'PS'. Field numbers 70-79.9 reserved for Radiology Nodes and X-ref 'RA'. Field numbers 720-725 reserved for DSSM Nodes and X-ref 'EC' and 'AEC'. Field numbers 740-749.9 reserved for QA Nodes and X-ref 'QA'. Field numbers 654-654.9 reserved for Social work Node 654 and X-ref 'SW'. Field numbers 500-500.9 reserved for mailman Node 500 and X-ref 'XM' and 'AXM'. Field numbers 740-749.9 reserved for QA Nodes and X-ref 'QA*'. Field numbers 910-910.9 reserved for Police Package Node and X-ref 'ESP'

Fileman ID: 200

Base VDM Class: New_Person-200
