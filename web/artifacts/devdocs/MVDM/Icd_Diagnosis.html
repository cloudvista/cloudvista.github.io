Icd_Diagnosis

Icd Diagnosis

ICD Diagnosis file #80 contains codes from the International Classification of Diseases (ICD) Clinical Modification (CM) provided by the Centers for Medicare and Medicaid Services (CMS) and the National Center for Health Statistics (NCHS). This file contains both the 9th (ICD-9-CM) and 10th (ICD-10-CM) Revisions.

If an entry needs to be added, modified or deleted, a patch will be issued containing the change.

This table file should NOT be edited in anyway by the site.
