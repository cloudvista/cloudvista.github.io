Request_Services

Request Services

This file allows services and specialties to be grouped in a hierarchy representing the sites services available. This grouping capability may be used with Review screens to filter out consults to a service, sub-service, specialty, or sub-specialty of consults/requests.

The main entry in this file is the "ALL SERVICES" entry. Other entries should be subordinate in its hierarchy.

The "ALL SERVICES" entry is used to display the hierarchy of the hospital services when the Clinician ordering a consult is prompted for "Select TO Service/Specialty:" to send the consult to.
