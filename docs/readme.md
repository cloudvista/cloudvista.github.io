2. docx: clean1 (standard title, remove rev history)
3. docx [upload] gdoc [clean2] (toc fonts:Arial)
4. gdoc [saveAs] html
5. gdoc [saveAs] docx
6. docx [writeage:saveAs] md
7. md [vscode] clean3 (find/replace)
